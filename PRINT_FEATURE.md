## Tips for Best Results

1. **Label Paper**: Use 2" x 1" or 3" x 1" label paper for best results
2. **Printer Settings**: 
   - Set paper size to match your labels
   - Disable margins for label printing
   - Use high quality/draft mode
3. **Page Setup**: In print dialog, select "More settings" and:
   - Set margins to "None" or "Minimum"
   - Enable "Background graphics" if needed
   - Select appropriate paper size

## Troubleshooting

**Print dialog doesn't open automatically?**
- Click the "Print Barcodes" button in the preview page
- Some browsers may block auto-print - check browser settings
- Try a different browser if auto-print is blocked

**Barcodes not showing?**
- Make sure JavaScript is enabled
- Check browser console for errors
- Refresh the page and try again

**Print quality issues?**
- Use high-quality printer settings
- Ensure printer has enough ink/toner
- Clean printer heads if needed
- Test print on regular paper first

## Manual Print Option

If the automatic print doesn't work:
1. The preview page will show all barcodes
2. Click the "Print Barcodes" button at the top
3. Or use keyboard shortcut: `Ctrl+P` (Windows) or `Cmd+P` (Mac)
